# XpressMobile Documentation

This repository contains documentation for the XpressMobile (XM) project at USXpress.

## Overview

XpressMobile is the driver-facing mobile application and backend system that enables USXpress drivers to:
- Manage loads and orders
- View payroll and settlement information
- Access training materials
- Submit documents and forms
- Communicate with dispatch
- Track performance metrics

## Documentation Structure

This repository will contain:
- Architecture documentation
- API documentation
- Feature specifications
- Deployment guides
- Troubleshooting guides
- Migration documentation
